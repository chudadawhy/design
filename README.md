# 2026 SNS Trends Guide 🚀

> **Live URL**: [chudadawhy.github.io/design](https://chudadawhy.github.io/design)

2026년 최신 SNS 플랫폼(Facebook, Instagram, YouTube, Threads, X) 트렌드를 시각화한 웹 가이드입니다.
초보자도 바로 참고할 수 있도록 특징, 핵심 기능, 타깃 유저, 운영 팁을 카드 형태로 제공합니다.

## 주요 기능

- 📊 **5대 SNS 플랫폼 분석** — 특징, 5대 핵심 기능, 타깃 유저, 초보자 운영 팁
- 🔄 **실시간 트렌드 수집** — GNews API 키 입력 후 버튼 클릭 시 최신 뉴스 수집
- 🎨 **2026 프리미엄 디자인** — 다크 모드 + 글래스모피즘 + 마우스 스포트라이트 효과
- 📱 **완전 반응형** — 모바일/태블릿/데스크톱 최적화

## 트렌드 수집 기능 사용법

1. [gnews.io](https://gnews.io/) 에서 무료 API 키 발급 (무료: 100건/일)
2. 사이트 상단 입력창에 API 키 입력
3. "🔄 트렌드 수집" 버튼 클릭
4. 각 SNS 카드 하단에 최신 뉴스가 자동으로 표시됩니다

## 기술 스택

- HTML5 / Vanilla CSS / Vanilla JavaScript
- Font Awesome 6.5 (아이콘)
- Google Fonts / Pretendard (폰트)
- GNews API (트렌드 뉴스 수집)
- GitHub Pages (호스팅)
